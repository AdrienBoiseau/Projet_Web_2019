Nos choix en matière de design :

Nous avons choisi un design sobre, il est parfois très simple et minimaliste. Il se base sur un template général présent sur tout le site. Ce template est constitué du header qui est en deux parties, la partie haute (en noir) est dynamique et affiche le nom de l'utilisateur connecté, et la partie du dessous qui permet la navigation dans le site. La deuxième partie du template est le footer qui contient le bouton a propos, et les copyright.

Concernant tout les formulaires présents sur le site, ils utilisent tous le même thème qui est défini dans les styles CSS.
